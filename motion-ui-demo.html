How to Implement

Three simple ways to use animations in your project:

1. CSS Classes (Simplest)

<div class="slide-transition-up pop-in"> This element slides up and pops in on page load </div>

2. Data Attributes (For scroll triggers)

<img class="fade-in" src="image.jpg" alt="Description"> <p data-scroll-animation="slide-up">Appears when scrolled into view</p>

3. JavaScript API (For complex interactions)

// Access the MotionUI object window.motionUI.bounceElement(element); window.motionUI.slideElement(element, 'up', 100, 400); window.motionUI.fadeElement(element, 300, true); window.motionUI.rotateElement(element, 360, 1000);
